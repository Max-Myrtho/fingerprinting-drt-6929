## Requirements

- Node.js **22.11.0** or higher

## Installation

Download and unzip the folder.
In the root directory, run the following commands:

```sh
npm install
npm run build
```

## Running the project

Start the preview server with:

```sh
npm run preview
```

Then, open the following link in your browser:
http://localhost:6929/
